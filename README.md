
# Task 15 - Customer Segmentation (RFM Analysis)

## Objective
Perform customer segmentation using RFM (Recency, Frequency, Monetary) Analysis.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Steps Performed
1. Data Cleaning
2. Removed cancelled invoices
3. Calculated Recency, Frequency, Monetary
4. Created RFM Scores using quantile binning
5. Segmented customers (Champions, Loyal, At Risk, Regular)
6. Visualized segment distribution
7. Exported final segmentation table

## Files Included
- task15_rfm.ipynb
- rfm_segments.csv
- segment_actions.txt
- README.md

## Outcome
Identified high-value customers and suggested business actions for each segment.
